# 🌤️ Hava Durumu Çekici

wttr.in sitesinden hava durumu bilgisini çeken küçük bir Python betiği.

## 🚀 Nasıl Çalışır?

1. Şehir adını gir
2. Betik internete bağlanır
3. wttr.in'den hava durumunu alır
4. Sonucu ekrana yazdırır ve `hava_kayit.txt` dosyasına kaydeder
